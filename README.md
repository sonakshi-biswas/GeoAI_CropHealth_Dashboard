# GeoAI Crop Health Dashboard – West Bengal NDVI Analysis

## Description
This project monitors crop health in West Bengal using **MODIS NDVI satellite data (2023)**. It provides:
- NDVI visualization
- District-level analysis
- NDVI anomaly for drought detection
- Interactive dashboard (click a district to see NDVI trend)

## Features
- Satellite NDVI monitoring
- NDVI anomaly detection for vegetation stress
- Vegetation classification (Red–Yellow–Green)
- District-level NDVI summary
- Interactive charts per district<img 


## Tech Stack
- Google Earth Engine JavaScript API
- Google Earth Engine UI

## How to Run
1. Open [Google Earth Engine Code Editor](https://code.earthengine.google.com/)
2. Copy the script `GeoAI_NDVI_WB.js`
3. Run the script – NDVI maps and interactive charts appear
